# Storage Cleaner
A simple, no-nonsense disk cleanup utility for Windows. Swipe through files Tinder-style to keep or delete them, visualize where your storage is going, and free up space without any bloat.
No internet connection. No ads. No updates. No paywall.
________________________________________________________________
## Features

Tinder-style swipe interface to review files
File previews for images, videos, and documents
Duplicate file detection
Folder size visualizer
Scheduled automatic scans
Summary screen with individual file restore before final deletion
_____________________________________________________________

## Installation
Download the installer and run it.
